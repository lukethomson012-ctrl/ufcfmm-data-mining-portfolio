# Data Mining Portfolio

This repository contains the group data mining portfolio for the UFCFMM module.

## Structure
- `notebook/` – Jupyter Notebook portfolio
- `data/` – Dataset or dataset source information
- `group/` – Administrative submission files

## How to run
Open `notebook/project_portfolio.ipynb` in Jupyter or VS Code with the Jupyter extension.
